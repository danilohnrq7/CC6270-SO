SYSTEM CALL: Interface que interliga nossos programas com os serviços do SO. Um serviço só pode ser executado pelo próprio SO 

Uma Sysrtem call é basicamente uma API, funções em C, C++... que se comunicam com nosso serviço
 Oferecem uma interface para os serviços disponíveis pelo SO. Geralmente, são funções escritas em C ou C++, algumas delas (baixo-nível) são escritas em assembly
